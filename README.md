# OCC-ops
Operations Control Center
Dashboard para gestionar flota y tripulaciones.

Como TCP con 5 años de experiencia, he desarrollado esta herramienta para digitalizar la visibilidad de operaciones diarias, permitiendo búsquedas rápidas de aeronaves y control de estados de tripulación bajo las FTL.

Tecnologías: HTML5, CSS3 (Modern UI), JavaScript ES6.
